Dataset introduction

The mtcars dataset in R is a built-in dataset that provides information about different models of cars from the 1970s. It includes various technical specifications and performance metrics for 32 cars, including attributes related to fuel consumption, horsepower, weight, and transmission.
I will use this dataset to create a hierarchical cluster analysis heatmap to provid insights into how specific car attributes co-occur across models.

Plot the heatmap

ha = columnAnnotation(d=anno_density(data.scale, which = "column",height = unit(1, "cm")))

partition <- Heatmap(data.scale, name = "mtcars heatmap ", 
        row_km = 2, column_km = 2,
        show_row_names = TRUE,
        row_dend_reorder = TRUE,
        row_names_gp = gpar(fontsize = 10),
        bottom_annotation= ha)

ht_inline<-Heatmap(mtcars[, c(8)], name = "Cylinder Config",
                   col = colorRamp2(c(0,1),c("white", "black")),
                   heatmap_legend_param = list(at = c(0,1),
                         labels = c("V-shaped", "Inline")),
                   width = unit(0.5,"cm"))

ht_manual_new <-Heatmap(mtcars[, c(9)], name = "Transmission",
                   col = colorRamp2(c(0,1),c("white", "grey")),
                   heatmap_legend_param = list(at = c(0,1),
                         labels = c("Automatic", "Manual")),
                   width = unit(0.5,"cm"))

draw(partition+ht_inline+ht_manual_new, auto_adjust = FALSE)

The cylinder configuration is labled as black:incline, white:V-shaped; The transmission is labelled as grey:manual, white:automatic.

Interpretation of the mtcars Cluster Heatmap

This cluster heatmap visualizes the mtcars dataset, focusing on relationships between car models and key performance metrics. The variables represented include mpg (miles per gallon), drat (rear axle ratio), qsec (1/4 mile time), wt (weight), hp (horsepower), cyl (cylinder count), and disp (displacement).

🔍 Key Observations

1. Hierarchical Clustering

  • Rows (Car Models): The hierarchical clustering groups car models based on similar patterns in their performance metrics. We observe two primary clusters:
    • Cluster 1 (Top) : Mostly high-performance, powerful cars like Chrysler Imperial and Lincoln Continental. These cars are characterized by high horsepower (hp), high weight (wt), and larger displacement (disp), shown in intense red colors.
    • Cluster 2 (Bottom) : This includes more fuel-efficient, lighter cars such as Fiat 128, Toyota Corolla, and Honda Civic, shown in blue tones, indicating lower hp, wt, and disp values.
  • Columns (Performance Metrics): Variables are also grouped into two main clusters: - Cluster 1 : Metrics related to fuel efficiency (mpg, drat, qsec). - Cluster 2 : Variables associated with engine power and size (hp, cyl, disp, wt), showing a strong relationship among these.

2. Color Coding

  • Heatmap Colors: The heatmap ranges from blue (low values) to red (high values).
    • High Values 🔴: Attributes like hp, cyl, wt, and disp appear in red, aligning with more powerful cars.
    • Low Values 🔵: Fuel-efficient metrics (mpg, drat, qsec) appear in blue, representing fuel-efficient models.

3. Categorical Variables

  • Cylinder Configuration (vs) : Indicates if the engine is V-shaped or inline. Inline engines are shown with darker shades, and V-shaped engines with lighter shades.
  • Transmission (am) : Shows cars with manual (black) and automatic (gray) transmissions. Many cars in the lower cluster have manual transmissions, while those in the upper cluster are automatic.

Conclusion

The heatmap highlights distinct clusters in the mtcars dataset, where certain cars share characteristics in weight, fuel efficiency, and engine power. It helps to quickly spot patterns, such as high-horsepower, V-shaped engine cars with automatic transmissions clustering together, separate from lighter, fuel-efficient, manual transmission vehicles.
